Introduction & Problem Statement

Introduction

  • Student feedback is critical for evaluating teaching quality and improving academic outcomes
  • CampusPulse is a browser-based feedback analytics platform requiring no server infrastructure
  • Automates collection, sentiment classification, and visual reporting of student feedback
  • Provides administrators with real-time, data-driven insights for academic decision-making

Problem Statement

  • Existing systems are manual, paper-based, and inconsistent
  • No real-time analysis or automated insights available
  • Administrators cannot efficiently process large feedback volumes
  • Delayed response to student concerns and grievances
  • No structured faculty-wise performance evaluation mechanism
Objectives

Primary Objectives

  • Develop a structured student feedback submission platform
  • Implement automated rule-based NLP sentiment analysis
  • Build a real-time admin analytics dashboard with visual charts
  • Enable faculty-wise performance tracking and CSV report export

Secondary Objectives

  • Role-based authentication for students and admins
  • 6-question structured evaluation questionnaire
  • AI-inspired insights and recommendations
  • Responsive design across all screen sizes
  • Student feedback history and profile statistics

Implementation Details

Landing Page

Hero section, animated sentiment cards, features overview, how-it-works, and role-based login cards

Authentication

Student registration/login and admin login with localStorage session management and role-based access

Feedback Form

Subject/faculty dropdowns, 5-star rating, 6-question questionnaire, category selection, and text input

Sentiment Engine

Weighted keyword dictionaries, negation detection, intensifier handling, and star rating weighting for classification

Admin Dashboard

Overview stats, Canvas pie chart, rating/category bar charts, faculty analysis grid with individual export buttons

CSV Export

Per-faculty reports with summary stats, questionnaire analysis, detailed records, and subject-wise breakdown
